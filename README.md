# 🎓 Smart Learn AI

**An AI‑Powered Learning Assistant** built with Python, Streamlit, and Groq AI.

Smart Learn AI helps students learn faster by combining an AI tutor, PDF analysis, quiz generation, and study planning – all in one sleek web app.

![Smart Learn AI Screenshot](screenshot.png) <!-- Add a screenshot later -->

---

## 🚀 Features

- **💬 AI Tutor** – Get instant explanations and examples on any topic.
- **📄 PDF Teacher** – Upload your notes and ask questions based on the content.
- **📝 Quiz Maker** – Generate practice quizzes with answers and explanations.
- **🧠 Explain Topic** – Get step‑by‑step breakdowns of complex subjects.
- **📅 Study Planner** – Build a daily revision schedule tailored to your needs.

---

## 🛠 Technology Stack

| Component   | Technology                              |
|-------------|-----------------------------------------|
| Frontend    | Streamlit                               |
| Backend     | Python 3.10+                            |
| AI Engine   | Groq API (Llama/Mixtral)                |
| PDF Parsing | PyPDF2                                  |
| Styling     | Custom CSS with glass‑morphism effects  |
